# Phishing Email Detection Tool (SOC Project)
This is a Python-based phishing detection tool designed for SOC analysts.
## Features
- Detects suspicious keywords
- Identifies malicious URLs
- Flags IP-based links
- Detects shortened domains
- Provides risk level (Low / Medium / High)
## Technologies
- Python
- Regex
- tldextract
- Colorama
## Sample Output
Shows risk level and indicators clearly for analysis.
